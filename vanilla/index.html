Editium

Text Editor

Getting Started

Install Editium in your project using your preferred method:

NPM

npm install editium

Yarn

yarn add editium

PNPM

pnpm add editium

unpkg CDN (Vanilla JS)

<link rel="stylesheet" href="https://unpkg.com/editium/vanilla/editium.css">
<script src="https://unpkg.com/editium/vanilla/editium.js"></script>

jsDelivr CDN (Vanilla JS)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/editium/vanilla/editium.css">
<script src="https://cdn.jsdelivr.net/npm/editium/vanilla/editium.js"></script>

Quick Usage (Vanilla JS)

const editor = new Editium({
  container: document.getElementById('editor'),
  placeholder: 'Start typing...',
  toolbar: 'all',
  height: '300px',      // or use number: 300
  minHeight: '200px',   // or use number: 200
  maxHeight: '400px',   // or use number: 400
  showWordCount: true,
  onChange: (data) => {
    console.log(data.html, data.json);
  }
});

Quick Usage (React)

import { Editium } from 'editium';

function App() {
  return (
    <Editium
      initialValue={[]}
      toolbar="all"
      placeholder="Start typing..."
      height="300px"
      minHeight="200px"
      maxHeight="400px"
      showWordCount={true}
      onChange={(html, json) => {
        console.log(html, json);
      }}
    />
  );
}
